Car Rental Website

Welcome to the Car Rental Website! This project allows users to register and login to the website. Users can view the available cars for rent, while admins have the ability to add new cars to the inventory. The website features include:

Registration with Regular Expression Validation

User and Admin Roles
Car Information Display
Admin Car Management (Add, Update, Delete)
User Authentication
MongoDB Database Integration
Express.js Backend
React Frontend
Node.js Server
Bootstrap Styling

Features

User Registration and Login: Users can register and log in to the website. Regular expression validation is used for secure and accurate user registration.
Admin and User Roles: The system differentiates between users and admins. Admins have additional privileges to manage car information.
Car Information Display: Users can view the available cars for rent. The website displays Car Name, Model Name, Company Name, Rent amount, and Capacity of the cars.
Admin Car Management: Admins can add new cars to the inventory, update car details, and delete cars from the list.
User Authentication: User authentication is implemented to ensure secure access to the website's features.

Technologies Used

MongoDB: The project utilizes MongoDB for storing car information and user details securely.
Express.js: The backend is built using Express.js, a robust Node.js framework for creating APIs and managing server-side logic.
React: The frontend is developed using React, a popular JavaScript library for building dynamic user interfaces.
Node.js: The project uses Node.js to run the server-side logic and interact with the database.
Bootstrap: The website is styled using Bootstrap, a widely-used CSS framework that provides responsive and sleek designs.

Getting Started

Clone the repository to your local machine.
Make sure you have Node.js and MongoDB installed.
Navigate to the project directory in the terminal.
Run npm install to install the project dependencies.
Start the server using the command node app.js.
Access the website through your browser at http://localhost:5000.

Admin Registration

To register as an admin, manually set the isAdmin field to true in the database. This will grant admin privileges to the registered user.

Usage
Register as a user using the website's registration form.
Log in using the provided credentials.
Explore the available cars for rent and view their details.
Admins can manage cars by adding, updating, and deleting car information.
